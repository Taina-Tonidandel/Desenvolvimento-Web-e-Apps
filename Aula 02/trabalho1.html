Aula 2- HTML Básico

HTML (HyperText Markup Language) é uma linguagem de marcação que define
a estrutura e o conteúdo de páginas da web. É a base de quase todas as páginas da internet.

  • Organiza o conteúdo visual
  • Fornece a base para outros recursos interativos
  • Determina as funções de todo o conteúdo visual de um site
